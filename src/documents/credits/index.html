Hi! My name is Sergey Chikuyonok and I’m creator of the Emmet toolkit.

I started project development under “Zen Coding” name back in 2009. Since then, the project has grown largely and has been warmly welcomed by thousands of users all over the world.

In 2012 the project was renamed. Emmet is going to be a brand name for a new tools that will help developers every day.
